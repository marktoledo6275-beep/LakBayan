# Top-Down RPG Starter

This project now includes a playable 2D top-down RPG starter for Unity.

## Included

- main menu scene
- playable `MainWorld` scene
- 4-direction top-down player movement
- mobile joystick and action button
- camera follow
- village elder NPC
- dialogue system
- first quest objective
- quest HUD
- top-down tileset asset

## One-Click Build

In Unity, click:

`LAKBAYAN > Build Playable Top-Down RPG Starter`

That will rebuild:

- `Assets/Scenes/MainMenu.unity`
- `Assets/Scenes/MainWorld.unity`

## How To Test

1. Open `Assets/Scenes/MainMenu.unity`
2. Press Play
3. Tap `START GAME`
4. Move the player
5. Talk to the elder
6. Walk to the shrine
7. Return to the elder

## Assets
